# ATS Sensex Stock Market Analysis Project

This project focuses on analyzing and predicting stock market trends using historical Sensex data and machine learning techniques. The objective is to build a data-driven system that processes financial time-series data, extracts meaningful features, and evaluates predictive models.

The project demonstrates how machine learning can be applied to financial datasets for trend analysis and decision support.

---

## Features

- Data cleaning and preprocessing of Sensex dataset  
- Exploratory data analysis and visualization  
- Feature extraction from time-series stock data  
- Implementation of machine learning models  
- Model evaluation using standard metrics  
- End-to-end notebook workflow  

---

## Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Jupyter Notebook
